**Employee Resource Tracking and Management System**

**Overview**

The Employee Resource Tracking and Management System is a centralized solution developed to automate employee resource management within an organization. It enables secure employee data management, profile status tracking, customer allocation, dashboard analytics, advanced search and filtering, and bulk Excel data synchronization. The application is available as both a web application and a Windows desktop application, providing flexibility for organizational use.


**Problem Statement**

Managing employee resource information through multiple Excel files is time-consuming, error-prone, and difficult to maintain. Manual updates often result in duplicate records, inconsistent data, and inefficient resource tracking. A centralized system is required to improve data accuracy, simplify management, and enhance operational efficiency.


**Proposed Solution**

The system provides a secure and centralized platform to manage employee information efficiently. It supports employee management, authentication, dashboard analytics, search and filtering, customer allocation, and bulk Excel uploads using a PostgreSQL database. The desktop version extends the same functionality for local organizational use.
